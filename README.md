## lispysnake2d

A simple, lightweight 2D engine for Lispy Snake, Ltd. graphical applications.
Provides basic SDL wrapping facilities to create cross-platform 2D games
and applications.

For now, entry-point hooks are not provided, i.e NDK/JNI.
